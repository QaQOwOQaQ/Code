# 类模板
格式：
template <class T1, class T2>
class A{
    T1 key;
    T2 value;
};

编译器由类模板生成类的过程叫类的实例化
由类模板实例化得到的类，叫**模板类**

在类模板初始化中，必须指明class的类型（函数模板无需指明）

函数模板可以作为类模板的成员

类模板的<参数列表项>中可以出现非类型参数
函数模板也可以，但如果函数模板出现了非类型参数，就必须初始化所有参数
template <class T, int size>
class A{};
